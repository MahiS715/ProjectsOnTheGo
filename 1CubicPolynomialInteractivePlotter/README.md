# 📈 Cubic Polynomial Interactive Plotter

An Interactive Streamlit Application to Explore Cubic Polynomials in Real Time.

---

## Features
- Adjust Coefficients a, b, c, d using Sliders
- Live Graph Update
- Root Visualization

---

## Mathematical Model

\[
y = ax^3 + bx^2 + cx + d
\]
